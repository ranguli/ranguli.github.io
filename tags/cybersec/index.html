Fixing uniscan.pl on Arch-based distros

10/04/2019
When trying to install the helpful (but seemingly abandoned) uniscan.pl script for pentesting, I found that it didn’t work straight out of the box with Manjaro. Here’s how I got around it: Find the location of your uniscan install ➜ which uniscan /bin/uniscan This returns /bin/bash/uniscan which cleverly isn’t an executable, it’s just: #!/bin/sh cd /usr/share/uniscan perl uniscan.pl "$@" So our uniscan script is in /usr/share/uniscan: ➜ /bin ls -l /usr/share/uniscan total 416 -rw-r--r-- 1 root root 5501 Apr 5 16:01 CHANGELOG.
